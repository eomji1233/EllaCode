Position Sticky

CSS Position의 속성값
        1. static, 아무것도 지정하지 않은 경우, default값으로 사용됨.
        - CSS의 Normal-flow에 따라 배치, offset이 적용되지 않는다.
        2. relative, 자신이 원래 있어야 할 위치의 offset에 따라 배치됨.
        * offset은 top, left, right, bottom 속성값이며 offset을 지정하지 않으면 
        static과 동일한 기능을 수행함. offset을 적지 않으면 static처럼 움직임.
        3. absolute, absolute로 정의된 요소는 CSS의 Normal-flow 흐름에서 배제됨.
        위치는 부모 요소(태그) 위치의 상대값에 따라 배치됨.
        * 단, 부모 요소가 static인 경우에는 조상 요소의 position 속성이 static이
        아닐 때까지 거슬러 올라가 배치됨. (지금같은 경우에는 올라가고 올라가서 body)
        
        4. fixed, 브라우저의 창에 상대적인 위치(offset)에 보이도록 함.
        5. sticky, 특정 조건에 고정되어 움직임.

화면 배치 변경 방법

박스(블록) 요소와 인라인 요소의 속성을 변경하여 배치방식을 변경할 수 있음

>> display:inline 테스트
인라인 요소로 변경되면서 width와 height 속성이 무시된다.

    지정되어 width와 height속성을 설정할 수 있다.
